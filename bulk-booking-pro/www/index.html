❓ Help & User Guide

🤖 AI Magic Scan Kaise Use Karein?

  • Parcel ki photo saaf (clear) kheechein.
  • Ensure karein ki Barcode (13 chars) aur VPP/COD amount photo mein padhne laayak ho.
  • AI sab kuch (Name, Address, Mobile, Pin, COD) khud bhar dega!

🎙️ Voice Mode Commands

Note: Voice Mode chalane ke liye HTTPS (Secure Netlify Link) zaroori hai.

  • Pehle kis bhi box par click karein aur bole jo type karna hai.
  • Next : Aagle box mein automatically jaane ke liye.
  • Delete it : Agar box khaali karna ho.
  • Replace X with Y : Galat shabd theek karne ke liye (e.g., "Replace Ganga with Banga").

⚡ Smart Features

  • Auto-Fallback API: 1st API Key ka quota finish hote hi app khud 2nd Key par chala jayega (Total 3000 Scans).
  • Auto-Save & Memory: Aapka data har second browser mein save hota hai. Purane PIN codes yaad rakhega.
  • Red Asterisk (*) : Ye wale boxes bharna zaruri hai.
